# Privacy Policy for UUCMS Calculator

**Last Updated: November 29, 2025**

## Data Collection

The "UUCMS Calculator" Chrome Extension **does not collect, store, or transmit any personal user data**.

## How it Works

- The extension operates entirely locally within your browser.
- It reads the marks displayed on the current web page (UUCMS result sheet) solely for the purpose of calculating the total, average, and percentage.
- This data is processed in real-time and is **never** sent to any external server or third party.

## Third-Party Services

This extension does not use any third-party analytics or tracking services.

## Contact

If you have any questions about this privacy policy, please contact the developer via the GitHub repository: https://github.com/2k3manu/uucms-calculator
